# SaaS Application – User Guide

## Introduction

This guide helps end-users navigate and use the SaaS Application effectively.

---

## Logging In

1. Open the application URL.
2. Enter your registered email and password.
3. Click **Login**.

If authentication fails, contact your system administrator.

---

## Dashboard Overview

The dashboard contains:

- Profile Summary
- Reports Section
- Notifications Panel
- Settings

---

## Generating Reports

1. Navigate to Reports.
2. Select the report type.
3. Choose date range.
4. Click Generate.
5. Download the report in PDF or Excel format.

---

## User Roles

| Role | Access Level |
|------|-------------|
| Admin | Full Access |
| Manager | Report & Team Access |
| User | Limited Access |
